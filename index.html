Domain Summaries
Domain 1

In Domain 1, we did a review about the basic HTML fundamentals, such as what goes in the head and body of a HTML document and how to make comments in HTML. We also learned how to use the script and noscript tag. The script tag is used to alert a user that the website/webpage uses JavaScript. The noScript would show up if the users' browser doesn't support Javascript. Lastly, we learned about what a meta tag is used for and how to add keywords into that tag.
