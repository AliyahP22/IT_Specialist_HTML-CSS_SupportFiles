Domain Summaries Domain 1

In Domain 1, we did a review about the basic HTML fundamentals, such as what goes in the head and body of a HTML document and how to make comments in HTML. We also learned how to use the script and noscript tag. The script tag is used to alert a user that the website/webpage uses JavaScript. The noScript would show up if the users' browser doesn't support Javascript. Lastly, we learned about what a meta tag is used for and how to add keywords into that tag.

Domain 2

Domain 2 was a good review of CSS techniques and tags. It went over how to create stylesheets and how to create a rule set. In addition, it went over a simple review on how to add color to paragraphs and even went over internal CSS and inline CSS. In addition to this, we were introduced to applying Pseudo Classes.

Domain 3

Domain 3 was a review on interactive tags and how to create them. We started domain 3 with a review on how to create a table and how and when to use hr and br tags. Then we dove a little deeper and did a review on the div tag and the span tag and how to use it. We then learn about summary and article tags. We also learn about the caption tags. Lastly, we learned about relative and absolute tags. Absolute takes you to a direct website while relative takes you to a different file.

Domain 4

Domain 4 teaches about multimedia tags such as video tags and tags for audio tags. We learn how to add a video to a webpage and how to add audio. We also learned how to use the track tag to add captions

Domain 5

Domain 5 is about more CSS properties. It reviews properties such as font styles, font weight, padding, and margins. It also demonstrates how to use the float tag and how to use relative and absolute positions It also teaches about overflows and static and fixed views

Domain 6

Domain 6 is all about reusing rule sets and browser safe fonts. It also does a review on how to make comments on a CSS file, how to resize text, and how to do tab order anad order the links in a certain order.
